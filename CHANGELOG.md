# Changelog

All notable changes to this project are documented here. Format loosely follows [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] — 2026-07-12
### Added
- Initial public release of the SmartHarvest AI frontend: Home, Prediction Dashboard, Model Comparison, and Documentation pages.
- Vanilla JS chart library (bar, horizontal bar, line, radar, donut, heatmap) with no external dependencies.
- Live/demo automatic fallback for the prediction API (`js/prediction.js`, `js/config.js`).
- District climate-risk heatmap for all 50 districts.
- Dark/light theme toggle, scroll-reveal animations, typing hero effect, magnetic buttons.
- PWA manifest, sitemap, robots.txt.

### Notes
- Model performance figures (RMSE/MAE/R²) are sourced from the training pipeline's `outputs/model_comparison.png` results.
