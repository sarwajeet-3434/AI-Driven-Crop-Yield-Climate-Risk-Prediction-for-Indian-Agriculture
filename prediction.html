How it works

Live model, with a demo fallback that never breaks the UI

01

Request sent

The form posts JSON to /api/predict on the URL set in js/config.js.

02

Live or fallback

If the backend responds within 4 seconds, its prediction is used. Otherwise the in-browser demo engine (js/prediction.js) takes over silently.

03

Same output shape

Both paths return identical fields, so connecting your real backend later requires zero frontend changes.
